Detailed Description

Descriptor and solver for a branched polymer species.

The block concentrations stored in the constituent Block objects contain the block concentrations (i.e., volume fractions) computed in the most recent call of the compute function.

The phi() and mu() accessor functions, which are inherited from PolymerTmp<Block>, return the value of phi (spatial average volume fraction of a species) or mu (chemical potential) computed in the last call of the compute function. If the ensemble for this species is closed, phi is read from the parameter file and mu is computed. If the ensemble is open, mu is read from the parameter file and phi is computed.
