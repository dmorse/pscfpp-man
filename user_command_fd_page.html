3.3.1 Commands for pscf_fd

3.3 Command Files (Prev)         3.3.2 Commands for pscf_pcNd (CPU) (Next)

Here is a example of a minimal command file for pscf_fd:

ITERATE in/w
WRITE_W out/w.bf
WRITE_C out/c.bf
FINISH

The command "ITERATE in/w.bf" instructs the program to read the file in/w.bf, which contains an initial guess for the chemical potential field, then iteratively solve the self-consistent field equations start from this initial guess. The command "WRITE_W out/w.bf" writes the resulting converged set of chemical potential fields to the file out/w.bf. The command "WRITE_C out/c.bf" writes the corresponding monomer volume fraction fields to the file out/c.bf. The script stops execution when it encounters the FINISH command.

The following table shows a list of available commands for pscf_fd:

command parameter(s) description
ITERATE filename [string] Read initial w field from file filename, iterate to convergence
WRITE_W filename [string] Write chemical potential (w) fields to file filename
WRITE_C filename [string] Write monomer volume fraction (c) fields to file filename
WRITE_BLOCK_C filename [string] Write monomer volume fractions for each block to file filename
COMPARE_HOMOGENEOUS mode [int] Compare solution to homogeneous solution(s)
WRITE_VERTEX_Q filename [string], polymerId[int], vertex[Id] Compare solution to homogeneous solution(s)
