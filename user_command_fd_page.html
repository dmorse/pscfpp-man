3.4.1 Commands for pscf_fd

3.4 Command Files (Prev)         3.4.2 Commands for pscf_pc (CPU) (Next)

Here is a example of a minimal command file for pscf_fd:

ITERATE in/w
WRITE_W out/w
WRITE_C out/c
WRITE_PARAM out/data
WRITE_THERMO out/data
FINISH

The command "ITERATE in/w" instructs the program to read the file in/w, which contains an initial guess for the chemical potential field, then iteratively solve the self-consistent field equations start from this initial guess. The command "WRITE_W out/w" writes the resulting converged set of chemical potential fields to the file out/w. The command "WRITE_C out/c" writes the corresponding monomer volume fraction fields to the file out/c. The command WRITE_PARAM writes the parameter file for this solution to the file out/data, and the subsequent command WRITE_THERMO appends thermodynamic data of the converged solution to the end of out/data. The script stops execution when it encounters the FINISH ommand.

The following table shows a list of available commands for pscf_fd:

command parameter(s) description
READ_W filename [string] Read w fields from file filename
ITERATE filename [string] (conditional) Iteratively solve SCFT equations (after reading initial w fields)
SWEEP Perform a sweep, as specified by the Sweep object in the param file.
WRITE_W filename [string] Write chemical potential (w) fields to file filename
WRITE_C filename [string] Write monomer volume fraction (c) fields to file filename
WRITE_BLOCK_C filename [string] Write individual block/solvent volume fraction (c) fields to file filename. The block c fields are given in the order that the blocks appear in the param file, followed by the solvent fields in the order that the solvents appear in the param file
WRITE_PARAM filename [string] Write param file associated with the current system. Sweep blocks are omitted.
WRITE_THERMO filename [string] Write thermodynamic data (free energy, pressure, phi, and mu for each species) to a file. If file already exists, this command will append thermodynamic data to the end of the file, rather than overwriting what is currently stored there.
COMPARE_HOMOGENEOUS mode [int] Compare solution to homogeneous solution(s)
WRITE_VERTEX_Q filename [string], polymerId [int], vertexId [int] Write propagator (q) field to a file for a specified vertex (vertexId) in a specified polymer (polymerId).
