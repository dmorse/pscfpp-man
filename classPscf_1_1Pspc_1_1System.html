Detailed Description

template<int D>
class Pscf::Pspc::System< D >

Main class in SCFT simulation of one system.

A System has (among other components):

  • a Mixture (a container for polymer and solvent solvers)
  • a Mesh
  • a UnitCell
  • a Basis (symmetry-adapted Fourier expansion basis)
  • an Iterator
  • Monomer chemical fields in both basis and grid formats
  • Monomer concentration fields in both basis and grid formats

Iteratively solve a SCFT problem.

template<int D>
void Pscf::Pspc::System< D >::compute ( )

Solve the modified diffusion equation once, without iteration.

This function calls the mixture().compute() function to solve the statistical mechanics problem for a non-interacting system subjected to the currrent chemical potential fields (wFields and wFieldRGrid). This requires solution of the modified diffusion equation for all polymers, computation of Boltzmann weights for all solvents, computation and setting of molecular partition functions for all species, and computation and setting of block and setting of concentration fields for all blocks and solvents, and computation of overall concentrations for all monomer types. This function does not compute the canonical (Helmholtz) free energy or grand-canonical free energy (i.e., pressure). Upon eturn, the flag hasCFields is set true.
