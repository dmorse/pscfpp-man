Detailed Description

template<typename Iterator, typename T>
class Pscf::AmIteratorTmpl< Iterator, T >

Template for Anderson mixing iterator algorithm.

Anderson mixing is an algorithm for solving a system of N nonlinear equations of the form g_{i}(x) = 0 for i = 0, ..., N-1, where x denotes a vector or array of unknown coordinate values. A vector or array of unknowns is referred here a "field", while a vector or array of values of the errors g_{0},..., g_{N-1} is referred to as a residual.

The type T is the type of the data structure used to store both field residual vectors.
