Detailed Description

template<int D>
class Pscf::Rpg::CFields< D >

A list of c fields stored in both basis and r-grid format.

This class is simply a named partial specialization of the base class template Pscf::Prdc::CFieldsTmpl, designed for use with a GPU. Please see documentation of the base class for API documentation.

void Pscf::Prdc::CFieldsTmpl< D, RField< D >, FieldIo< D > >::setWriteUnitCell ( UnitCell< D > const & cell)

Set unit cell used when writing field files.

This function creates a stored pointer to a UnitCell<D> that is is used by the writeBasis and writeRGrid functions, which each write the unit cell parameters from in this object to a field file header. This function may only be called once.
