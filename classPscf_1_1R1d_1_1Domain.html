bool Pscf::R1d::Domain::isShell ( ) const
inline

Is this a cylindrical or spherical shell?

This value is relevant only if the geometry mode is spherical or cylindrical. If so, isShell is set true if the optional parameter xMin is present and assigned a positive value in the parameter file. If geometryMode is planar or xMin is absent, then isShell is false.
