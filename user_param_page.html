3.2 Parameter Files

3.1 Input File Types (Prev)         3.3 Command Files (Next)

The parameter files required by the different PSCF programs use a similar syntax, but contain somewhat different contents. The following three sections document parameter file formats for different programs. The documentation of the parameter file format for the pscf_fd 1D finite difference program includes a discussion of elements that are used for all programs, and thus should be read first. The discussion of the file format for pscf_pcNd focuses on parts of the file format that differ from that required for pscf_fd. The file formats for the pscf_pcNd and pscf_pgNd programs are almost identical, reflecting the fact that these are closely analogous programs designed to run on different hardware.
