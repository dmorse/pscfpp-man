Detailed Description

template<int D>
class Pscf::Rpg::Domain< D >

Spatial domain for a periodic structure with real fields, on a GPU.

See the interface of the Prdc::DomainTmpl base class template for complete API documentation. The Rpg::Domain class template is basically a named partial specialization of the base class template, defined using template type parameters FFT = Prdc::Cuda::FFT<D>, WLT = Prdc::Cuda::WaveList<D>, and FIT = Rpg::FieldIo<D> . The public interface is identical to that of the base class.
