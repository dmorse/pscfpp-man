Pspc::Iterator

Pscf::Pspc::Iterator<D> is a base class that defines the general behavior that is expected of any SCFT iterator used in the pscf_pc programs. In the param file, a subclass of Iterator must be included as a subblock of the System block.

There are two different types of iterators available for use with the pscf_pc programs, as described in the table below. For a description of the parameters format for a particular iterator, follow the links in the table.

Pspc::AmIterator Anderson Mixing iterator. Uses the Anderson Mixing algorithm to solve the SCFT equations.
Pspc::AmIteratorFilm Thin Film Anderson Mixing iterator. Uses the Anderson Mixing algorithm to solve the SCFT equations, while imposing a mask (geometric confinement) and an external field (wall/polymer chemical interactions) that simulate a polymer system that is confined to a thin film geometry.
