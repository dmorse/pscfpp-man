Detailed Description

template<int D>
class Pscf::Rpg::AverageListAnalyzer< D >

Analyze averages and block averages of several real variables.

This class evaluates the average of several sampled real variables, and optionally writes block averages to a data file during a simulation. It is intended for use as a base class for Analyzers that evaluate averages and (optionally) block averages for several physical variables.
