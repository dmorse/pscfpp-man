template<int D>
void Pscf::Pspc::Block< D >::computeConcentration ( double  prefactor)

Compute concentration (volume fraction) for block by integration.

This should be called after both associated propagators are known. Upon return, grid point r of array cField() contains the integal, int ds q(r,s)q^{*}(r,L-s) times the prefactor parameter, where q(r,s) is the solution obtained from propagator(0), q^{*}(r,s) is the solution of propagator(1), and s is a contour variable that is integrated over the domain 0 < s < length(), where length() is the block length. The "prefactor" parameter should be set to prefactor = phi/(L q), where phi is the overall volume fraction for this molecular species, L is the number of monomers in the species, and q is the species partition function, i.e., the spatial average of q(r,L). This function is called by Polymer<D>::compute().
