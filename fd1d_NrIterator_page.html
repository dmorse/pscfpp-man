Fd1d::NrIterator

A Pscf::Fd1d::NrIterator<D> object is an iterator that solves the SCFT equations using the Newton-Raphson method. The maximum number of iteration steps is 100, and the lattice parameters are held constant during iteration. The only user-defined parameter is the error tolerance.

The parameter file format for an NrIterator object is:

NrIterator{
epsilon real
}

Parameters are described below:

epsilon Error tolerance.

Iteration stops when the scalar error is less than epsilon. The total error is defined in NrIterator as the maximum component of a residual vector (the L-∞ norm), in which each element of the vector represents a residual define on a node of the spatial grid.
