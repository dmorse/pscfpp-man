template<int D>
void Pscf::Pspc::FieldIo< D >::readFieldHeader ( std::istream &  in,
int &  nMonomer,
UnitCell< D > &  unitCell 
) const

Reader header of field file (fortran pscf format)

This reads the common part of the header for all field file formats. This contains the dimension of space, the unit cell, the group name and the the number of monomers. The unit cell data is read into the associated UnitCell<D>, which is thus updated.

This function throws an exception if the values of "dim" and "N_monomer" read from file do not match values of D and the input parameter nMonomer, respectively. The group name is not checked.

Parameters
ininput stream (i.e., input file)
nMonomerexpected value of nMonomer
unitCellassociated crystallographic UnitCell<D>
checkMonomerdepending on field type, check monomer number or not
