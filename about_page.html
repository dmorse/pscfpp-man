PSCF is a package of software for solving the Edwards-Helfand self-consistent theory for polymer liquids. The version presented here is written primarily in C++, with code for use on graphics processing units (GPUs) written in Cuda. This C++/Cuda version of PSCF is still under development, but is intended to eventually supersede the older Fortran program of the same name.

Some differences in design relative to the existing Fortran code for periodic structure, and the expected advantages of the new code are as follows:

  • A more flexible design has allowed implementation of several different SCFT solvers that are designed for use in different geometries or that use different algorithms or hardware, within a common framework.
  • A description of polymer topology that allows simulation of mixtures containing arbitrarily complex acyclic branched block copolymers and homopolymers, in addition to the linear chains allowed by the Fortran code.
  • An object oriented C++ design will allow the creation of different objects to represent different phases, which we expect to facilitate analysis of phase coexistence in mixtures.
  • The transition to C++ has simplified implementation of Cuda code to use nVidia GPUs.
