Fd1d::FdIterator

An FdIterator solves is an iterator that uses the simple relaxation method presented by F. Drolet and G.H. Fredrickson to solve the SCFT equations of a 1D problem with only two monomer types, such as an AB block copolymer or A/B polymer blend. The iteration algorithm will halt if applied to a system with more than two types of monomer.

The parameter file format for an FdIterator object is:

FdIterator{
epsilon real
lambdaPlus real
lambdaMinus real
maxIterations int
}

Parameters are described below:

epsilon Error tolerance.
lambdaPlus relaxation parameter for W_{+} (field conjugate to total concentration).
lambdaMinu relaxation parameter for W_{-} (field conjugate to composition).
maxIterations maximum number of iterations attempted.

Iteration stops when the scalar error is less than epsilon. The total error is defined in FdIterator as a root-mean-square of errors at individual grid pointw.
