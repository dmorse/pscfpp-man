System

A Pspc::System<D> object is the main object in a SCFT computation for a D dimensionally periodic structure using a pseudo-spectral algorithm on CPU hardware.

See also
Pscf::Pspc::System

The parameter file format is:

System{
Mixture{ ... }
Interaction{ ... }
Domain{ ... }
< Iterator > { ... }
< Sweep >* { ... }
}

Here, as elsewhere, optional parameters or subblocks are indicated by an asterisk, conditional elements by a dagger, and polymorphic elements (for which the label can correspond to the label of any of several subclasses) are indicated by enclosing the generic or base class name in angle brackets.

The purpose of each of the main subblocks is discussed below.

Mixture{ .... } Description of system composition. See Mixture docs.
Interaction{ .... } Flory-Huggins excess free energy. See Interaction docs.
Domain{ .... } Discretization and unit cell. See Domain docs.
< Iterator > { .... } Polymorphic Iterator block, for any one of several subclasses of Pspc::Iterator.
< Sweep >* { ... } Optional polymorphic Sweep block, for any one of several subclasses of Pspc::Sweep
