3.5 Field Files

3.4 Examples (Prev)         4 Developer Information (Next)

Each PSCF program can read and write files that contain chemical potential and monomer volume fraction fields. Several different file formats are used to represent these fields and other quantities that can be expressed mathematically as functions of one or more spatial coordinates. The pscf_pcNd and pscf_pgND CPU and GPU programs, which are both designed to treat periodic microstructures, both use file formats that were originally created for the older Fortran PSCF program. The pscf_fd1d program for one-dimensional programs uses a simpler file format in which values of fields specified at nodes of a regular one-dimensional grid. Different field file formats are described in the pages linked below.
