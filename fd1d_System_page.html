Fd1d::System

A Pscf::Fd1d::System <D> object is the main object in SCFT computation using a finite difference algorithm to treat a one-dimensional problem.

See also
Pscf::Fd1d::System

The parameter file format is:

System{
Mixture { ... }
Interaction { ... }
Domain { ... }
< Iterator > { ... }
< Sweep >* { ... }
}

Here, as elsewhere, optional parameters or subblocks of a parameter file block are followed by an asterisk, and polymorphic elements (for which the label can correspond to the label of any of several subclasses) are indicated with the generic or base class name enclosed in angle brackets.

Explanations and links to documentation for enclosed subblocks and parameters:

Mixture { .... } Description of mixture composition. See 3.3.4 Mixture Block docs
Interaction { .... } Flory-Huggins excess free eenrgy. See Interaction docs
Domain { .... } Spatial domain and discretization. See Fd1d::Domain docs
< Iterator > { .... } Polymorphic block for any of several subclasses of Fd1d::Iterator
< Sweep > * { ... } Optional polymorphic Fd1d::Sweep block. If present, define a sequence of states along a line through parameter space, which can be solved using continuation to construct initial guesses.
